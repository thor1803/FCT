Change the footer credit text for Storefront theme.

== Description ==
[Demo](http://quadmenu.com/storefront/) | [Storefront Mega Menu](https://wordpress.org/plugins/quadmenu-storefront)

This plugin allows you to easily edit and change the footer credit and copyright text in the Woocommerce Storefront theme through the wordpress admin panel.

Works only with [Storefront Theme](https://wordpress.org/themes/storefront).

== Installation ==

1. Go to the Plugins Menu in WordPress
1. Search for "Storefront Footer Text"
1. Click "Install"
